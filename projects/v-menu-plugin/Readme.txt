Magento theme

Is made with the base of Magento blank which is inside of default theme.
Both folders contents the path structure of the place where to install them.
In Dashboard/configuration/Design is where to choose theme default, template blank
and skin blank.

Wordpress plugin vMenu

Copy v-menu-plugin folder inside wp-content/plugin. In dashboard goes to plugins and activate.
This plugin is to be used with sidebar and shows regions, cities and villages of Ukraine.

Know issues:
- Open database with all information was not reached, it uses a mockup data base.
- Menu has a delay when is clicked.
